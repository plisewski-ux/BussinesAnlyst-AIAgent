Wywiad procesowy

Przeprowadzę z Tobą anonimowy wywiad (20 pytań), na podstawie którego wygeneruję kartę procesu biznesowego i schemat BPMN. Nie zbieramy żadnych danych osobowych.

⚠️ Brak skonfigurowanego klucza API. Przejdź do zakładki Ustawienia i uzupełnij dane.

Pomiń wywiad i przejdź od razu do generowania dokumentów na podstawie gotowego tekstu. Przydatne przy testach i ponownej analizie.

🔑 Połączenie z API OpenAI

Klucz API jest przechowywany wyłącznie w pamięci przeglądarki (localStorage). Nie jest nigdzie przesyłany poza wywołaniami do OpenAI.

Generuj na: platform.openai.com → API Keys
Polecane: gpt-4o (jakość) lub gpt-4o-mini (szybkość/koszt)

💬 Prompt wywiadu

System prompt agenta prowadzącego wywiad. Możesz edytować zasady i styl rozmowy. Zmienne wypełniane automatycznie: {Q20} — lista 20 pytań, {ROLE} — stanowisko rozmówcy, {DEPT} — dział.

Instrukcja systemowa {Q20} {ROLE} {DEPT}

📄 Prompt analizy — Karta procesu

System prompt do generowania JSON karty procesu. Schemat JSON jest dołączany automatycznie. Zmienne: {TRANSCRIPT} — transkrypt wywiadu, {ROLE}, {DEPT}, {DATE}.

Instrukcja systemowa (schema JSON dołączana automatycznie) {TRANSCRIPT} {ROLE} {DEPT} {DATE}

📊 Prompt analizy — Schemat BPMN

System prompt do generowania struktury BPMN. Schemat JSON jest dołączany automatycznie. Zmienne: {PROCESS_NAME}, {STEPS}.

Instrukcja systemowa (schema JSON dołączana automatycznie) {PROCESS_NAME} {STEPS}
